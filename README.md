# DoubleStoreTaskList

## Project Setup (Docker)

```sh
docker-compose build
docker-compose up
```

## Project Setup (manual)

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
